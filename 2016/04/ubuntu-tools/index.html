ubuntu-tools

发表于 2016-04-01

中午瞌睡的时候 不小心把 /etc目录给删掉了 没有办法拷别人的etc 只好重装
顺便记一下必备的工具

1. chrome

从官网下载安装包

2. sogou pinyin

从官网下载安装包 如果没装fcitx 要先装fcitx

all setttings -> language support -> keyboard input method system
选择fcitx 然后 重启

3. sublime-text

从这里可以找到Package Control的安装脚本
上次发现的超酷炫的theme

4. pycharm

一个很强大的py ide 下载地址在这里

然后发现个可以注册lience的地方

5. vim

配置文件 ~/.vimrc 还好已经备份

6. ssh

直接拷贝 .ssh 文件夹里的两个key
或者重新生成新的

6. git 没什么可多说的

7. terminator

还有个tmux大家都在用 有空也研究一下

7. java

跟c/c++一样 有一些软件得有java环境才能运行

还有JAVA_HOME这个环境变量
